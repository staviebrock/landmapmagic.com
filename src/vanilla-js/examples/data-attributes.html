Declarative map initialization using HTML data attributes

Iowa Farm - Soil Analysis

Shows SSURGO soil data for an Iowa farm. Click on soil areas to see detailed information.

Nebraska Field - Crop Monitoring

Displays CDL crop data for a Nebraska field with field boundaries available.

Kansas Ranch - Property Lines

Illinois Farm - Complete Analysis

Comprehensive view with all datasets available. Toggle layers using the legend.

HTML Code

<!-- Iowa Farm - Soil Data -->
<div id="iowa-farm" 
     data-landmap-api-key="your-api-key"
     data-landmap-center="-93.5,42.0"
     data-landmap-zoom="10"
     data-landmap-layers="ssurgo,plss"
     data-landmap-visible="ssurgo"
     data-landmap-legend="true"
     data-landmap-click-info="true"
     style="width: 100%; height: 400px;">
</div>

<!-- LandMapMagic Script -->
<script src="https://static.landmapmagic.com/js/landmap-vanilla-latest.min.js"></script>
<script>
    // Auto-initialize all maps with data attributes
    LandMapMagic.autoInit().then(maps => {
        console.log('Initialized', maps.length, 'maps');
    });
</script>

Available Data Attributes

data-landmap-api-key      Your LandMapMagic API key (required)
data-landmap-center       Map center as "lng,lat" (default: "-98.5795,39.8283")
data-landmap-zoom         Initial zoom level (default: 4)
data-landmap-layers       Available layers as comma-separated list (default: "ssurgo,cdl,plss,clu")
data-landmap-visible      Initially visible layers (default: none)
data-landmap-legend       Show legend - "true" or "false" (default: "true")
data-landmap-click-info   Show click info - "true" or "false" (default: "true")
